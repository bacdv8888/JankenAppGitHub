
# じゃんけんゲーム (Janken Game)

これは、ReactとTailwind CSSで構築された、モダンでスタイリッシュなじゃんけんゲームです。プレイヤーはコンピューターと対戦することができます。

## ✨ 主な特徴

- **直感的なUI**: 洗練されたデザインとスムーズなアニメーションで、快適なゲーム体験を提供します。
- **コンピューターとの対戦**: コンピューターがランダムに手を出し、プレイヤーと勝負します。
- **スコア追跡**: プレイヤーとコンピューターのスコアがリアルタイムで表示されます。
- **ゲームリセット**: いつでもスコアをリセットして、新しいゲームを開始できます。
- **レスポンシブデザイン**: デスクトップからスマートフォンまで、様々なデバイスで快適にプレイできます。

## 🛠️ 技術スタック

- **React**: UIの構築に使用されているJavaScriptライブラリです。
- **TypeScript**: 静的型付けにより、コードの堅牢性と開発効率を向上させています。
- **Tailwind CSS**: ユーティリティファーストのCSSフレームワークで、迅速なUI開発を可能にしています。

## 📂 プロジェクト構成

```
.
├── index.html                # アプリケーションのエントリーポイントとなるHTMLファイル
├── index.tsx                 # ReactアプリケーションをDOMにマウントするメインファイル
├── App.tsx                   # ゲームの主要なロジックとUIを含むメインコンポーネント
├── components/               # 再利用可能なReactコンポーネント
│   ├── HandButton.tsx        # プレイヤーが手を選ぶためのボタン
│   ├── HandIcon.tsx          # グー・チョキ・パーを日本語で表示するアイコン
│   ├── ResultDisplay.tsx     # 勝敗結果を表示するコンポーネント
│   └── Scoreboard.tsx        # スコアボードを表示するコンポーネント
├── constants.ts              # ゲーム内で使用される定数（手の種類、結果メッセージなど）
├── types.ts                  # TypeScriptの型定義（Hand, Resultのenum）
└── metadata.json             # プロジェクトのメタデータ
```

## 🚀 起動方法

このプロジェクトはViteを使用しているため、以下の手順で起動できます。

1. プロジェクトの依存関係をインストールします。
   ```bash
   npm install
   ```
2. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
3. ターミナルに表示されたURL（例: `http://localhost:3000`）をブラウザで開きます。

---

また、このプロジェクトはビルドステップを必要とせず、モダンなブラウザで直接 `index.html` を開くことでも実行できます。
ただし、ESモジュールの制約により、ローカルサーバー経由で実行することが最も確実です。

1.  プロジェクトファイルをローカルにダウンロードします。
2.  ターミナルを開き、プロジェクトのルートディレクトリに移動します。
3.  以下のいずれかのコマンドを実行して、ローカル開発サーバーを起動します。

    **Node.js がインストールされている場合 (npx):**
    ```bash
    npx serve
    ```

    **Python 3 がインストールされている場合:**
    ```bash
    python -m http.server
    ```

4.  ターミナルに表示されたURL（例: `http://localhost:3000` や `http://localhost:8000`）をブラウザで開きます。

## 遊び方

1.  画面下部に表示される「グー」「チョキ」「パー」のいずれかのボタンをクリックして、あなたの手を選びます。
2.  あなたの選択後、コンピューターの手がランダムに選ばれ、画面に表示されます。
3.  勝敗結果が表示され、スコアが更新されます。
4.  「もう一度」ボタンをクリックすると、次のラウンドが始まります。
5.  スコアボードの「RESET」ボタンをクリックすると、スコアがリセットされ、全く新しいゲームを始めることができます。
